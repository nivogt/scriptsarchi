# Scripts jArchi (perso)
Personal repository for jArchi scripts used with Archi.

## Structure

| Folder | Purpose |
|---|---|
| `customs/ai/` | AI-assisted generation scripts |
| `customs/exports/` | Extract / export data from a model |
| `customs/miscellaneous/` | Utility and transformation scripts |
| `customs/model/` | Create or generate model elements |
| `examples/` | Reference scripts shipped with jArchi |
